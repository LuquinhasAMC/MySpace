Linguagens

HTML

O HTML é uma linguagem de marcação, ela é basicamente o esqueleto de um site. É ela que construímos os componentes da interface de um site, como botões, títulos e etc.

CSS

O CSS é uma linguagem de estilização, ela serve para estilizar os componentes HTML.

JavaScript

O JavaScript é uma linguagem de programação, ela serve para criar a lógica por trás de um site, permitindo que o usuário interaja com os componentes.

FrameWorks

Electron

O Electron é um FrameWork para JavaScript que permite o desenvolvimento de aplicações desktop com tecnologia web, como HTML e CSS.

Capacitor

O Capacitor é um FrameWork para JavaScript que permite desenvolver aplicativos para celulares com tecnologia web, como HTML e CSS.
